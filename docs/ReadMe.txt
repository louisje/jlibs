To add new Tiddlter "MyTiddler"
first add following line to MainMenu.wiki
    [[MyTiddler]]
now open wiki.html, and click MyTiddler hyperlink
click [edit] link. type following text:
    <<tiddler "MyTiddler.wiki">>
click [done]
click [backstage] and then click [save]
